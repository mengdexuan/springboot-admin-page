利用p6spy拦截并查看数据库执行操作


P6SPY提供了如下几个功能：
	记录SQL语句的执行时间戳。
	记录SQL语句类型
	记录SQL填入参数的和没有填入参数的SQL语句
	根据配置的时间控制SQL语句的执行时间，对超出时间的SQL语句输出到日志文件中
